# Retreat Shirt Types

Configure the shirts offered to servers and walkers in this retreat: name, color, sizes, and who they apply to. If your role is read-only you'll see the list, but the editing buttons will be hidden.

## Add a type

Under **Add type** you enter:

- **Name** (e.g. "White with pink") and **Color** using the picker.
- **Order** in which it appears in lists.
- **Required for walkers** and **Optional for servers** (toggle them by tapping).

## Available sizes

Type a size and press **Enter** or comma to add it as a chip; tap the chip's X to remove it. You can also apply a quick **preset** (México, Colombia, or Internacional).

## Edit and delete

Each configured type is edited in its card: change name, color, order, sizes, and the toggles. Press **Save** to apply changes or **Delete** to remove it (irreversible, asks for confirmation).
