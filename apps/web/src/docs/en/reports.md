# Reports

This section explains the available reports in the Emaus system.

## Overview

The reports section provides various ways to view and export data about your retreat. Reports can be viewed on-screen, printed, or exported to Excel/PDF.

## Available Reports

### Participant Reports

#### Walker List

- Complete list of all walkers
- Includes contact information
- Shows assignment status
- Exportable to Excel

#### Server List

- Complete list of all servers
- Shows roles and assignments
- Contact information included

#### Family Groups Report

- Shows family relationships
- Color coding visualization
- Helps with avoiding conflicts

### Assignment Reports

#### Table Assignments Report

- Shows all tables with leaders and walkers
- Indicates table capacity
- Highlights conflicts

#### Bed Assignments Report

- Room-by-room assignments
- Shows bed types
- Unassigned beds listed
- Age-based assignment summary

### Financial Reports

#### Payment Summary

- Total payments received
- Breakdown by payment method
- Outstanding balances

#### Scholarship Report

- Scholarships awarded
- Budget remaining
- Recipient information (confidential)

### Inventory Reports

#### Inventory Status

- Current inventory levels
- Items checked out to walkers
- Missing/damaged items

#### Walker-to-Item Ratio

- Ensures proper inventory levels
- Based on walker count

## Exporting Reports

All reports can be exported in multiple formats:

1. **Excel (.xlsx)**
   - Full data export
   - Enables further analysis
   - Compatible with most spreadsheet software

2. **PDF**
   - Formatted for printing
   - Professional appearance
   - Suitable for distribution

3. **CSV**
   - Simple text format
   - Compatible with many systems
   - Smaller file size

## Custom Reports

### Creating Custom Views

1. Navigate to the data view (Walkers, Servers, etc.)
2. Use filters to narrow data
3. Show/hide columns as needed
4. Click **Export Current View**

### Saving Report Templates

Save frequently used report configurations:

1. Set up your filters and column selections
2. Click **Save as Template**
3. Name the template
4. Access saved templates from the reports menu

## Scheduling Reports

Set up automated report generation:

1. Navigate to Settings â†’ Reports
2. Click **Schedule Report**
3. Choose report type
4. Set frequency (daily, weekly)
5. Select recipients
6. Save the schedule

## Report Permissions

Not all users have access to all reports:

- **Viewers**: Can view basic reports only
- **Coordinators**: Can generate reports for their retreats
- **Admins**: Full access to all reports
- **Superadmins**: Can access financial and sensitive reports
