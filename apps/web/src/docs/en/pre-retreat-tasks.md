# Pre-Retreat Tasks

This section explains how to use the "what to do and when" checklist **before** a retreat: the logistics that must be prepared months or weeks in advance (parish, retreat house, speakers, snacks, flowers, etc.).

📺 **[Watch the tutorial (2 min)](https://youtu.be/pPguV-Gg7Bs)**

## Overview

Pre-Retreat Tasks replace the paper spreadsheet. Each task has:

- A **time before the retreat** (e.g. "4 months before", "2 weeks before"), from which the **due date** is calculated automatically based on the retreat start date.
- A **status**: Pending, In progress, Done, or Not applicable.
- A **responsible person**, which can be a retreat server and/or a free-text name.
- Optional **sub-tasks**: a main task can have several steps, each with its own status.

## Importing the checklist

The first time, import the tasks from a template:

1. Go to **Logistics → Pre-Retreat Tasks**.
2. Click **Import from template**.
3. Choose the template (default "Pre-retiro — Emaús") and the mode:
   - **Only add missing**: keeps what you already captured and adds the template's new tasks.
   - **Replace all**: deletes the current tasks and recreates them from the template.

You can also enable automatic creation when creating the retreat, in the **Logistics** tab of the retreat form.

## Tracking progress

- Use each task's **checkbox** to mark it Done.
- Checking a **main task** also checks all its active sub-tasks. When all active sub-tasks are checked, the main task checks itself.
- Tasks marked **"Not applicable"** don't count toward progress.

## Responsible people

- Tap **"Unassigned"** (or the responsible's name) to open the search box and assign a retreat server instantly.
- You can also type a free-text name from the task edit dialog.

## Status colors and dates

Each task shows an indicator based on its due date:

- **Overdue** (red): the date has passed and it's not done.
- **Soon** (amber): due within 7 days.
- **On time** (gray): still ahead.
- **Done** (green): completed.

A **"● Today"** line separates overdue from upcoming tasks, so you can see at a glance where things stand.

## Filter, search and export

- **Search** by task name, responsible, or note.
- **Filter** by status: All / Pending / Overdue / Done.
- Group **by time** or view the **flat list**.
- **Export** downloads a CSV file (opens in Excel) with all tasks and sub-tasks.

## Global template

Under **Global Settings → Pre-Retreat Task Template** you can edit the master checklist: create your own templates, set the default, and add/edit tasks and sub-tasks (sub-tasks can inherit the parent's timing). Template changes apply to future retreats that import it; retreats that already materialized it keep their tasks.
