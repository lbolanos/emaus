# Navigation Architecture

## System Sections

The system is divided into two types of sections:

### Retreat Sections
These sections depend on a selected retreat. The retreat selector appears in the header so you can choose which retreat to work with.

- **Main**: Retreat dashboard, flyer
- **People**: Walkers, servers, partial servers, waiting list, canceled
- **Assignments**: Tables, rooms, user type and table, bed assignments, responsibilities
- **Financial**: Payments
- **Reports**: Bags report, medicines report, badges
- **Services**: Palancas, notes and meeting points, food, cancellation and notes, inventory, message templates
- **Administration**: Role management

### Global Sections
These sections do not depend on a specific retreat. The retreat selector is hidden and the current section title is shown instead.

- **Community**: Communities, members, meetings, admins, templates
- **Social**: Profile, search users, friends, followers, testimonials, my retreats
- **Global Settings**: Houses, telemetry, global templates, inventory items, change password, help

## Retreat Selector Behavior

- The selector only appears when you navigate to a section that requires a retreat
- Your retreat selection is preserved when navigating between sections
- When changing retreats, permissions are automatically updated
- If no retreats are available, a button to create a new one is shown
