# Getting Started

Welcome to the Emaus Retreat Logistics Management System. This guide will help you understand the basics of navigating and using the system.

## System Overview

The Emaus system is designed to help manage religious retreats efficiently. It provides tools for:

- **Participant Management**: Track walkers (caminantes) and servers (servidores)
- **Housing Assignments**: Manage room and bed assignments
- **Table Assignments**: Organize seating arrangements with leaders
- **Payment Tracking**: Monitor payments and scholarships
- **Reporting**: Generate reports and export data

## Navigation

The main navigation is located on the left sidebar. Click on any section to access its features:

- **Dashboard**: Overview of current retreat status
- **Walkers**: Manage retreat participants
- **Servers**: Manage retreat service team members
- **Tables**: View and manage table assignments
- **Bed Assignments**: Manage room assignments
- **Payments**: Track payment status
- **Reports**: Access various reports
- **Settings**: Configure system options

## Getting Help

Click the **?** button in the header (next to the language switcher) to access contextual help for the page you're currently viewing.

For full documentation, navigate to the Help section from the sidebar or the quick help panel.
