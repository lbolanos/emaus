# My Retreats

The My Retreats feature allows you to view all retreats you have attended and revisit the special memories from each experience.

## Viewing Your Retreats

1. Navigate to **My Retreats** in the sidebar (social section)
2. View all retreats you have participated in, organized into two sections:
   - **Shared Memories**: Retreats with photos and/or music added
   - **Retreats Without Memories**: Retreats that haven't been documented yet

## Retreat Memory Cards

Each retreat card displays:

- **Parish Name**: Location where the retreat was held
- **Date Range**: Start and end dates of the retreat
- **Retreat Type Badge**: men, women, couples, or effeta
- **Memory Photo**: Group photo or special moment (if added)
- **Music Playlist**: Link to the retreat's playlist (if added)

## Retreat Types

- **Men/Hombres**: Men's retreats
- **Women/Mujeres**: Women's retreats
- **Couples/Parejas**: Couples' retreats
- **Effeta**: Effeta retreats

> **Note**: Only retreats where you were registered as a participant will appear in this list.
