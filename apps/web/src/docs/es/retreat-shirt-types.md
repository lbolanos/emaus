# Tipos de playera del retiro

Configura las playeras que se ofrecerán a servidores y caminantes en este retiro: nombre, color, tallas y a quién aplican. Si tu rol es de solo lectura verás la lista, pero los botones de edición estarán ocultos.

## Agregar un tipo

En **Agregar tipo** capturas:

- **Nombre** (ej. "Blanca con rosa") y **Color** con el selector.
- **Orden** en el que aparece en las listas.
- **Requerido para caminantes** y **Opcional para servidores** (actívalos tocándolos).

## Tallas disponibles

Escribe una talla y pulsa **Enter** o coma para agregarla como chip; toca la X del chip para quitarla. También puedes aplicar un **preset** rápido (México, Colombia o Internacional).

## Editar y eliminar

Cada tipo configurado se edita en su tarjeta: cambia nombre, color, orden, tallas y las casillas. Pulsa **Guardar** para aplicar los cambios o **Eliminar** para borrarlo (acción irreversible, pide confirmación).
