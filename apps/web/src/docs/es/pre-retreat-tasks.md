# Tareas Pre-Retiro

Esta sección explica cómo usar el checklist de "qué hacer y cuándo" **antes** de un retiro: las tareas de logística que hay que preparar con meses o semanas de anticipación (parroquia, casa, charlistas, snacks, flores, etc.).

## Descripción General

Las Tareas Pre-Retiro reemplazan la hoja de cálculo en papel. Cada tarea tiene:

- Un **tiempo antes del retiro** (por ejemplo "4 meses antes", "2 semanas antes"), del que se calcula automáticamente la **fecha límite** a partir de la fecha de inicio del retiro.
- Un **estado**: Pendiente, En curso, Lista o No aplica.
- Un **responsable**, que puede ser un servidor del retiro y/o un nombre escrito a mano.
- **Sub-tareas** opcionales: una tarea principal puede tener varios pasos, cada uno con su propio estado.

## Importar el checklist

La primera vez, importa las tareas desde un template:

1. Entra a **Logística → Tareas Pre-Retiro**.
2. Pulsa **Importar desde template**.
3. Elige el template (por defecto "Pre-retiro — Emaús") y el modo:
   - **Solo agregar faltantes**: mantiene lo que ya capturaste y agrega las tareas nuevas del template.
   - **Reemplazar todo**: borra las tareas actuales y las vuelve a crear desde el template.

También puedes activar la creación automática al crear el retiro, en la pestaña **Logística** del formulario de retiro.

## Marcar el avance

- Usa la **casilla** de cada tarea para marcarla como Lista.
- Si marcas una **tarea principal**, se marcan también todas sus sub-tareas activas. Si marcas todas las sub-tareas activas, la tarea principal se marca sola.
- Las tareas en **"No aplica"** no cuentan para el progreso.

## Responsables

- Toca **"Sin asignar"** (o el nombre del responsable) para abrir el buscador y asignar un servidor del retiro al instante.
- También puedes escribir un nombre libre desde el modal de edición de la tarea.

## Semáforo y fechas

Cada tarea muestra un indicador según su fecha límite:

- **Vencida** (rojo): la fecha ya pasó y no está lista.
- **Próxima** (ámbar): vence en 7 días o menos.
- **A tiempo** (gris): aún falta.
- **Lista** (verde): completada.

Una línea **"● Hoy"** separa las tareas vencidas de las próximas, para ver de un vistazo dónde vamos.

## Filtrar, buscar y exportar

- **Buscar** por nombre de tarea, responsable o nota.
- **Filtrar** por estado: Todas / Pendientes / Vencidas / Listas.
- Agrupar **por tiempo** o ver la **lista corrida**.
- **Exportar** descarga un archivo CSV (se abre en Excel) con todas las tareas y sub-tareas.

## Template global

En **Configuración Global → Template Tareas Pre-Retiro** puedes editar el checklist maestro: crear plantillas propias, marcar la predeterminada, y agregar/editar tareas y sub-tareas (las sub-tareas pueden heredar el tiempo del padre). Los cambios al template aplican a los retiros futuros que lo importen; los retiros que ya lo materializaron conservan sus tareas.
