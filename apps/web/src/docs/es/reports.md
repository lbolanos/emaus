# Reportes

Esta sección explica los reportes disponibles en el sistema Emaus.

## Descripción General

La sección de reportes proporciona varias formas de ver y exportar datos sobre su retiro. Los reportes pueden verse en pantalla, imprimirse o exportarse a Excel/PDF.

## Reportes Disponibles

### Reportes de Participantes

#### Lista de Caminantes

- Lista completa de todos los caminantes
- Incluye información de contacto
- Muestra el estado de asignación
- Exportable a Excel

#### Lista de Servidores

- Lista completa de todos los servidores
- Muestra roles y asignaciones
- Información de contacto incluida

#### Reporte de Grupos Familiares

- Muestra relaciones familiares
- Visualización de código de colores
- Ayuda a evitar conflictos

### Reportes de Asignaciones

#### Reporte de Asignaciones de Mesas

- Muestra todas las mesas con líderes y caminantes
- Indica la capacidad de la mesa
- Resalta conflictos

#### Reporte de Asignaciones de Camas

- Asignaciones habitación por habitación
- Muestra tipos de cama
- Camas sin asignar listadas
- Resumen de asignaciones por edad

### Reportes Financieros

#### Resumen de Pagos

- Total de pagos recibidos
- Desglose por método de pago
- Saldos pendientes

#### Reporte de Becas

- Becas otorgadas
- Presupuesto restante
- Información de beneficiarios (confidencial)

### Reportes de Inventario

#### Estado de Inventario

- Niveles de inventario actuales
- Artículos prestados a caminantes
- Artículos faltantes/dañados

#### Relación Caminante-Artículo

- Asegura niveles de inventario adecuados
- Basado en la cantidad de caminantes

## Exportar Reportes

Todos los reportes se pueden exportar en múltiples formatos:

1. **Excel (.xlsx)**
   - Exportación completa de datos
   - Permite más análisis
   - Compatible con la mayoría de software de hojas de cálculo

2. **PDF**
   - Formateado para impresión
   - Apariencia profesional
   - Adecuado para distribución

3. **CSV**
   - Formato de texto simple
   - Compatible con muchos sistemas
   - Tamaño de archivo más pequeño

## Reportes Personalizados

### Crear Vistas Personalizadas

1. Navegue a la vista de datos (Caminantes, Servidores, etc.)
2. Use filtros para acotar datos
3. Muestre/oculte columnas según sea necesario
4. Haga clic en **Exportar Vista Actual**

### Guardar Plantillas de Reporte

Guarde configuraciones de reporte usadas frecuentemente:

1. Configure sus filtros y selecciones de columnas
2. Haga clic en **Guardar como Plantilla**
3. Nombre la plantilla
4. Acceda a plantillas guardadas desde el menú de reportes

## Programar Reportes

Configure la generación automatizada de reportes:

1. Navegue a Configuración → Reportes
2. Haga clic en **Programar Reporte**
3. Elija el tipo de reporte
4. Configure la frecuencia (diario, semanal)
5. Seleccione destinatarios
6. Guarde el programa

## Permisos de Reportes

No todos los usuarios tienen acceso a todos los reportes:

- **Visores**: Solo pueden ver reportes básicos
- **Coordinadores**: Pueden generar reportes para sus retiros
- **Administradores**: Acceso completo a todos los reportes
- **Superadministradores**: Pueden acceder a reportes financieros y sensibles
