# Palancas

📺 **[Ver video tutorial](https://youtu.be/5mUo2Q1v3Ws)**

Las **palancas** son las cartas y mensajes de apoyo que reciben los caminantes durante el retiro. Esta pantalla te ayuda a llevar el control de cuántas se pidieron, cuántas llegaron y quién las coordina para cada caminante.

## Qué muestra la tabla

Para cada caminante ves su número, nombre y datos de referencia como la **fecha del último pago** y el **conteo de mensajes**. Estos campos son de solo lectura y sirven de contexto.

## Qué puedes registrar

Los campos editables por caminante son:

- **Coordinador de palancas**: quién se encarga de reunir sus palancas.
- **Palancas solicitadas**: cuántas se pidieron a familiares y amigos.
- **Palancas recibidas**: cuántas ya llegaron.
- **Notas de palancas**: observaciones libres (pendientes, acuerdos, recordatorios).

## Cómo usarla

Edita los datos directamente en la tabla o abriendo el formulario del caminante, y ve comparando **solicitadas** contra **recibidas** para detectar a quién le faltan palancas y darle seguimiento antes del retiro.
