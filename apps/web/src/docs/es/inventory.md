# Inventario

Aquí administras los **suministros del retiro**: qué se necesita, cuánto hay y qué falta comprar. Cada retiro parte del catálogo global de artículos y tú ajustas cantidades, cajas y estados a medida que avanza la logística.

## Cantidades y recalcular

Con **Recalcular** el sistema calcula la cantidad requerida de cada artículo según los caminantes (inscritos o esperados) y el ratio del catálogo. La columna **Actual** la editas tú; la columna **Comprar** muestra el faltante en rojo. Un banner arriba resume los **items insuficientes**.

## Agregar y configurar items

- **Agregar item**: elige artículos del catálogo global o crea uno **nuevo solo para este retiro**.
- **Config. para este retiro** (en el menú ⋮ de cada fila): sobreescribe el ratio, fija una cantidad o **excluye** el artículo, sin tocar el catálogo global.
- Marca el **estado del ciclo** de cada item: Pendiente, Empacado, En sitio, Consumido o Devuelto. También puedes asignar una **Caja** y notas.

## Filtros, catálogo y más acciones

Busca por nombre, equipo o caja; agrupa por categoría, equipo, caja o estado; y usa las **columnas** que quieras ver. Desde **Más acciones**: copiar inventario de un retiro anterior, imprimir la **lista de empaque** o la **lista de compras**, importar/exportar Excel y ver el **historial de cambios**.

## Catálogo global (Artículos de Inventario)

En **Configuración → Artículos de Inventario** mantienes el catálogo maestro que comparten todos los retiros: nombre, categoría, equipo, unidad, **ratio por caminante** o cantidad fija, y si el artículo está activo. Editar un artículo aquí afecta a todos los retiros que lo usen.
