# Primeros Pasos

Bienvenido al Sistema de Gestión Logística de Retiros Emaus. Esta guía le ayudará a entender los conceptos básicos de navegación y uso del sistema.

## Descripción General

El sistema Emaus está diseñado para ayudar a gestionar retiros religiosos de manera eficiente. Proporciona herramientas para:

- **Gestión de Participantes**: Seguimiento de caminantes y servidores
- **Asignaciones de Alojamiento**: Gestionar asignaciones de habitaciones y camas
- **Asignaciones de Mesas**: Organizar los arreglos de asientos con líderes
- **Seguimiento de Pagos**: Monitorear pagos y becas
- **Reportes**: Generar reportes y exportar datos

## Navegación

La navegación principal se encuentra en la barra lateral izquierda. Haga clic en cualquier sección para acceder a sus funciones:

- **Panel**: Resumen del estado actual del retiro
- **Caminantes**: Gestionar participantes del retiro
- **Servidores**: Gestionar miembros del equipo de servicio
- **Mesas**: Ver y gestionar asignaciones de mesas
- **Asignaciones de Camas**: Gestionar asignaciones de habitaciones
- **Pagos**: Seguimiento del estado de pagos
- **Reportes**: Acceder a varios reportes
- **Configuración**: Configurar opciones del sistema

## Obtener Ayuda

Haga clic en el botón **?** en el encabezado (junto al selector de idioma) para acceder a la ayuda contextual para la página que está viendo actualmente.

Para la documentación completa, navegue a la sección Ayuda desde la barra lateral o el panel de ayuda rápida.
